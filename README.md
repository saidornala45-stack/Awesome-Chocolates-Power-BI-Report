This project showcases a professional Power BI dashboard built to analyze chocolate shipment data across suppliers, products, regions, and order statuses. The report provides clear KPI tracking, trend analysis, performance insights, and interactive visuals to support business decision-making.


## Project Overview

The dashboard transforms raw shipment data into actionable insights by analyzing:

- Monthly shipment trends

- Revenue and box volumes

- Delivered vs Cancelled orders

- Supplier and product performance

- Regional shipment distribution

The project includes data cleaning, data modeling, DAX measures, and visual design following BI best practices.
## Dataset Description
The dataset contains the following fields:
| Column       | Description                     |
| ------------ | ------------------------------- |
| ShipmentID   | Unique shipment identifier      |
| SPID         | Supplier ID                     |
| PID          | Product ID                      |
| GID          | Region ID                       |
| Shipdate     | Shipment date                   |
| Amount       | Revenue generated               |
| Boxes        | Number of boxes shipped         |
| Order_Status | Delivered / Cancelled / Pending |
## Key Dashboard Features

- KPI Cards: Revenue, Shipments Count, Boxes Shipped, Delivery %

- Trend Analysis: Monthly shipment & revenue trends

- Product & Supplier Performance: Top performers and comparisons

- Order Status Breakdown: Delivered vs Cancelled orders

- Regional Analysis: Shipments by geographic region

- Interactive Filters: Product, supplier, region, and date-based slicing
## Data Transformation & Modeling

- Cleaned and formatted data using Power Query

- Built calculated columns for date and categorical grouping

- Created DAX measures for KPIs (Total Revenue, Total Boxes, Delivery Rate, etc.)

- Established relationships to enable accurate and dynamic filtering
## Insights Generated

- Identified top products and suppliers contributing most revenue

- Found regional trends affecting box volume and performance

- Observed seasonal shipment fluctuations

- Highlighted cancellation patterns and supplier reliability issues
